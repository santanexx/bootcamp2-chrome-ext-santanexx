Bootcamp Helper — Extensão MV3

Extensão do Bootcamp II. Esta página serve de landing: explica instalação e fornece links para download.

Instalação manual (modo desenvolvedor)

  1. Abrir chrome://extensions e ativar Developer mode.
  2. Clicar em Load unpacked e selecionar a pasta do projeto (a raiz onde está o manifest.json).
  3. Testar o popup e o content script nas páginas definidas.

Download

Link para o arquivo .zip estará na Release do repositório (ou faça o ZIP da pasta).

Permissões

  • storage
  • Host permission: https://*/* (exemplo de uso mínimo)
