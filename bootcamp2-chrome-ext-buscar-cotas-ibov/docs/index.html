Buscar cotas IBOV

Extensão Chrome (Manifest V3) — busca nome, logo e preço atual de uma cota (BRAPI)

Instalação manual

  1. Clone ou baixe o repositório.
  2. Acesse chrome://extensions no Chrome e ative Developer Mode.
  3. Clique em Load unpacked e selecione a pasta do projeto (a raiz onde está manifest.json).

Uso

Abra o popup da extensão e digite a sigla (ex: PETR4) e clique em Buscar. A extensão exibirá o nome, logo e preço atual.

Download

Ao executar a pipeline (GitHub Actions) será gerado dist/extension.zip como artefato / Release.
